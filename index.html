ADDU Nation — Alumni Portal

A single-page alumni community platform for Ateneo de Davao University, built with AngularJS 1.x. This started life as a Figma prototype — "Making Stories into Screens" — and was carried all the way through to a fully built, working web app. Alumni can log in, donate to causes, access academic documents, browse career opportunities, and manage their profile — with a separate admin side for managing submissions, campaigns, and moderation queues. It's built as a full PWA (Progressive Web App) with a service worker and manifest.json, so it's installable like a native app, fully responsive across devices, and still works offline once loaded. This is a pure frontend showcase — there's no backend or database behind it, so nothing you enter is actually saved or persisted. Try it live below, or open it full-screen.

Web Development · Data Viz
Nmap Visualizer & Log Analyzer

A frontend dashboard built to take raw, complex network scan data and make it actually readable. Drop in an Nmap XML (-oX) or JSON scan log and it's parsed entirely client-side — no backend — into an interactive executive dashboard: a force-directed network map (vis.js) clustering hosts by risk level, Recharts breakdowns of top open ports and risk distribution, live search/filter/sort across the host table, and a detail drawer with full port and service data per host. Built in React, with a react-window-virtualized table and memoized filtering/sorting so large scans stay smooth instead of re-rendering the whole page on every keystroke. No sample data on hand? Use the "Load sample scan" button inside the demo to populate it with a synthetic ~40-host subnet scan. Try it live below, or open it full-screen.

Web Development · AI
Student Enrollment System

A full-stack enrollment system built with an HTML/CSS/JavaScript frontend and a Python backend, backed by a hybrid database setup — an SQL database for structured records like student profiles, course offerings, and enrollment transactions, and MongoDB for more flexible, document-based data such as grade encoding and evaluation results. On top of the standard enroll/view/manage workflow, it includes an AI-assisted recommendation feature that processes student data (like completed courses and standing) to suggest suitable subjects or sections automatically, instead of requiring manual advising for every case.

Unlike the browser-based projects above, this one isn't something that can just run off a static file — it depends on a Python backend and a live database connection, so there's no way to embed a working version directly on this site. Rather than show static screenshots that only capture one screen at a time (and wouldn't really demonstrate how the backend, database, and AI logic connect), the full project — Python files, database schema, and frontend — is packaged as a ZIP so anyone interested can download it, look through the actual code, and run it locally themselves.
